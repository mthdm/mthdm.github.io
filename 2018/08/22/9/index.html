hexo-NEXT主题添加来必立评论

发布于 2018-08-22 正经的 HEXO  HEXO 

　　最近我由于抛弃了wordpress的博客改用了hexo，虽然也是加了不少插件，但还是感觉少点什么，就在今天上午我突然想了起来—评论区！！
　　我也在百毒上找了一些添加评论的教程，例如友言，网易云跟帖之类的，但大多不是停止服务需要备案就是设置过程太复杂，有的甚至没有QQ登录！！终于我找到了来必立，这个只需要修改一行代码，还可以使用QQ，微信，新浪，百度，人人网等账号登录，而且评论后台也做的非常好（真-站长推荐）。
　　首先打开来必立官网https://livere.com/
　　点击右上角的小人来注册一个账号（需要使用邮箱验证）
　　注册好后点击安装（没错就是安装）

复制下图红色括号以内的内容

将其粘贴到next的配置文件中(_config.yml)如下图所示

如果看不清就看下面的代码行（第７行）

# Support for youyan comments system.
# You can get your uid from http://www.uyan.cc
#youyan_uid: your uid

# Support for LiveRe comments system.
# You can get your uid from https://livere.com/insight/myCode (General web site)
livere_uid: MTAyMC8zOTEyNi8xNTY1Mw

# Gitment
# Introduction: https://imsun.net/posts/gitment-introduction/
# You can get your Github ID from https://api.github.com/users/<Github username>

记得要吧livere前面的#去掉，还有就是冒号后面有一个空格（我就忘输了一次）
　　然后保存退出，重新部署看看是不是文章下多了评论框（づ￣3￣）づ╭❤～

　　另外一个我非常推荐这玩意的原因就是这个后台非常好用，可以统计评论信息-具体看图

　　如果还有什么地方不明白可以Q我哦ヾ(๑╹◡╹)ﾉ”
　　不过这次我非常推荐直接在评论区留言，在评论区留言，在评论区留言，重要的事情说三遍！！！

　　还有最后一件要说的事…..我26号就开学了，以后更新频率肯定会下降ヽ(`Д´)ﾉ
